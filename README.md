## iOSアプリの主要機能
優先度順↓
1. サインイン機能（ユーザ情報とプランター情報を紐付ける）
2. 現在のプランター情報（湿度や水量）を見れる機能
3. 間引きガイド機能（とりあえずS3に格納された画像の表示のみ）
4. 水やりガイド機能（山田くんの論文データをもとにその日の水やり量についてプッシュ通知）
--------------------------------------------------------------------------
5. スケジュール管理機能（カレンダー）
6. SNSっぽい機能（プロフィール、チャット、検索）
